Setup Instructions

  1. Go to notion.so/my-integrations
  2. Click "+ New integration"
  3. Name it "Meetio"
  4. Select your workspace
  5. Copy the "Internal Integration Token"
  6. Create or open your transcripts database in Notion
  7. Click "Share" → Add your integration
  8. Copy the database ID from the URL
  9. Paste both values above and save
